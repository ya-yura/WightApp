# Килограмм

PWA-калькулятор цены за килограмм для быстрого сравнения товаров по весу и стоимости.

## Деплой

Приложение публикуется через GitHub Pages:

https://ya-yura.github.io/WightApp/

## Функции

- расчет цены за килограмм по цене и весу;
- расчет цены или веса по двум известным значениям;
- сохранение и сравнение товаров в локальной истории;
- PWA manifest и service worker для установки и offline-кеша.
